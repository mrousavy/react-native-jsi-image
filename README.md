# react-native-jsi-image

A writeable in-memory Image JSI Host Object

## Installation

```sh
npm install react-native-jsi-image
```

## Usage

```js
import { multiply } from "react-native-jsi-image";

// ...

const result = await multiply(3, 7);
```
